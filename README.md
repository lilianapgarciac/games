# Proyecto: TerrestGame

## 1. Project Description:

 Este es mi primer videojuego utilizando la librería Pygame dirijido a un público para beginners.  El proyecto está desarrollado con el lenguaje de programación Python. 
 El juego se trata de evitar 

## 2.  Tech Stack:

  - Python 3
  - Pygame
    
## 3. Funcionalidades

  - Correr
  - Saltar
  - Menú de Inicio y Salida.
  - Desaparecer cuando es atrapado por el enemigo.
  - Botón de reinicio.

## 4. Instalación:

  1. Clonar el repositorio de github.
  
  ```
  https://github.com/lilianapgarciac/games.git
  ```
  
  2. Abrir una nueva carpeta con el IDE de tu preferencia.
  
  3. Explorar.
  
  4. Abrir el archivo open_game.py y ejecutarlo.
  
  5. Y listo. ¡Empieza a jugar!
